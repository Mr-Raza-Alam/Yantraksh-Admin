## 📖 About
This repository contains the administrative frontend architecture for the Yantraksh Tech Fest. It provides festival organizers and module leads with a robust, secure interface to manage events, participants, and operational logistics effectively.

## 🛠️ Tech Stack
- **Framework:** React 18
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Routing:** React Router v7
- **HTTP Client:** Axios
- **Linting:** ESLint 9

## ✨ Features
- **Centralized Dashboard:** A comprehensive view of fest operations and vital metrics.
- **Modern UI:** Built from the ground up with Tailwind CSS and sleek Lucide icons for an intuitive interface.
- **Type-Safe Ecosystem:** Written entirely in TypeScript for predictable state flow and early error-catching.
- **Optimized Performance:** Packaged with Vite for instant server start and lightning-fast Hot Module Replacement (HMR).

## ⚙️ Setup Instructions
1. Navigate into the project directory: `cd Yantraksh-Admin`
2. Install dependencies: `npm install`
3. Start the local Vite development server: `npm run dev`
4. Build for production context: `npm run build`
